# Simple NTP client

## Usage

```console
$ ntpdate --help
Simple NTP client

Usage: ntpdate [OPTIONS] <SERVER>...

Arguments:
  <SERVER>...  Host name or IP address of the server

Options:
  -o <version>      Specify the NTP version to send, which can be 1, 2, 3 or 4 [default: 4]
  -t <seconds>      Specify the maximum time waiting for a server response, in seconds and fraction [default: 2]
  -h, --help        Print help information
```
